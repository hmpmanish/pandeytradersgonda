# Pandey Traders (Sukhdev Pandey Khad Beej Bhandar)

Welcome to the **Pandey Traders** repository! This is a complete, production-ready MERN stack application built for managing inventory, showcasing products, and more.

## 🚀 Features

- **Full MERN Stack**: Built with MongoDB, Express.js, React (Next.js), and Node.js.
- **Admin Dashboard**: Full CRUD capabilities for products, categories, and more.
- **Image Management**: Integrated with Cloudinary for seamless multiple image uploads.
- **Authentication**: Secure JWT-based login with protected routes.
- **Responsive Design**: Optimized for Mobile, Desktop, and Tablet.
- **Performance Optimized**: Built for speed, SEO, and scalability using Next.js.

## 🛠️ Tech Stack

- **Frontend**: Next.js (React), TailwindCSS, Axios
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Atlas)
- **Authentication**: JSON Web Tokens (JWT), bcryptjs
- **Media Storage**: Cloudinary

## 📂 Folder Structure

```
pandey-traders/
├── client/          # Next.js Frontend Application
│   ├── public/      # Static assets
│   ├── src/         # Application source code
│   └── .env.example # Frontend environment variables
└── server/          # Node.js Express Backend
    ├── config/      # Database and integration configurations
    ├── controllers/ # API logic
    ├── models/      # Mongoose schemas
    ├── routes/      # Express routes
    └── .env.example # Backend environment variables
```
